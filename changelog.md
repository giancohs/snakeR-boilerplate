# Changelog

## [0.1.1] - 2026-03-12

### Added
- Updated devcontainer.json to include Desktop Lite feature. This way we can use the desktop from the container and debug selenium scrapers, instead of relying on the host machine, which can cause issues and bugs (like not opening the chrome browser from the container). In order to use this feature, we need to open a chrome windows, and open the address: `http://localhost:6080`. 
- Reorganized the project structure and folders.
- Improved the install-dev-tools.sh script to install Chrome and ChromeDriver, and document the versions installed.
- Added a changelog.md file to track the changes made to the project boilerplate.
- Added a docs-and-references folder to document the installed versions of the dependencies.
